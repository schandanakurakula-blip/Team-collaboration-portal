# Team Collaboration Portal (HTML-Based)

## Overview
This project is a web-based portal that simulates a SharePoint-like collaboration platform for managing team content and communication.

## Features
- Announcements section  
- Document management section  
- Quick access links  
- Structured layout for better user experience  

## Technologies Used
- HTML  
- CSS  

## Use Case
This project demonstrates content management and collaboration features similar to internal tools used in organizations.
